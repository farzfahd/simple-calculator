# REKOMENDASI URUTAN MEMBACA

## Untuk Pemula (Fundamental):
1. Start dengan Khan Academy Videos + Buku Fundamental (dari list)
2. Fokus: Pengertian konsep, bukan memorisasi
3. Contoh jalur: 
   - Math: Stewart Calculus
   - Physics: Halliday-Resnick
   - Chemistry: Brown's Chemistry

## Untuk Menengah (SMA/Kuliah Awal):
1. Lanjut ke buku Menengah
2. Kombinasikan dengan practice problems
3. Contoh jalur:
   - Math: Linear Algebra (Lay) + Differential Equations
   - Physics: Griffiths Electrodynamics
   - Chemistry: Brown + Organic Chemistry (Klein)

## Untuk Lanjutan (Master/Advanced):
1. Pilih 2-3 buku lanjutan per topik
2. Kombinasikan dengan research papers
3. Join study groups atau seminar

## Untuk Ahli (Research/PhD):
1. Focus pada research papers (ArXiv, Journal)
2. Buku ahli sebagai reference
3. Bimbingan dari advisor/mentor
